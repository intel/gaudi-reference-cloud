Cloud Monitor
#############

Use Cloud Monitor to identify bottlenecks, performance issues, or troubleshoot network or I/O traffic on |ITAC| services.
Run Cloud Monitor to visualize performance metrics.

Choose an option below.

Bare Metal and VM Cloud Monitor
*******************************

Manage and monitor your compute instances.

* Try :ref:`cloud_monitor_bm_vm`

|IKS| Cloud Monitor
*******************

Manage the control plane of your implementation of |IKS|.  Track API servers, monitor Kubernetes* clusters, or troubleshoot performance.

* Try :ref:`cloud_monitor_iks`
