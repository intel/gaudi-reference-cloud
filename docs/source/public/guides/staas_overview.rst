Manage Storage
##############

|ITAC| storage solutions serve developers' needs for:

* Scalability
* Accessibility
* Low-latency
* Throughput

:ref:`Object Storage <staas_object>`

* Typically includes unstructured data such as photos, videos, emails, webpages, sensor data, and audio files.

**Benefits**:

* Ideal for developing cloud native applications
* Developers easily analyze any object in the bucket by functional characteristics
* Access from a single, virtual storage repository
* Near-infinite scalability

:ref:`File Storage <staas_file>`

* Typically includes large repositories, development environments, media stores, or user directories.

**Benefits**:

* Access to shared files across multiple systems
